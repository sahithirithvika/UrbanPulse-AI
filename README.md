# 🌍 UrbanPulse AI

> **Turning Citizen Voices into Smarter Community Decisions**

UrbanPulse AI is an AI-powered civic intelligence platform that transforms citizen complaints into actionable insights. It enables authorities to identify priority issues, monitor community trends, and make data-driven decisions through an interactive dashboard.

Built as a prototype for the **Google Gen AI Academy APAC Hackathon** under the theme **"AI for Better Living and Smarter Communities."**

---

## 📌 Problem Statement

Communities generate a large number of civic complaints every day, including road damage, garbage overflow, water leakage, traffic congestion, and public safety concerns. These complaints are often handled manually, resulting in delayed responses and inefficient resource allocation.

UrbanPulse AI addresses this challenge by organizing, prioritizing, and visualizing community issues, enabling authorities to respond faster and improve public services.

---

## 💡 Solution

UrbanPulse AI provides a centralized platform where users can report community issues and authorities can monitor them through an intelligent dashboard.

The system analyzes complaints, categorizes them, assigns priority levels, generates executive insights, and displays interactive visualizations that support faster and smarter decision-making.

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/sahithirithvika/UrbanPulse-AI.git
cd urbanpulse-ai
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
